# 공황 및 불안 장애 대처 서비스 제공 애플리케이션

## 목표
- 공황에 대처할 수 있는 AI 에이전트 학습
- 에이전트와 사용자를 연결할 수 있는 네트워크 구성
- 사용자 데이터를 저장할 수 있는 DB 구성
- 사용자 데이터를 분석할 수 있는 환경 구성
   
## AI Agent
공황을 겪는 사용자 전문 음성 상담 AI 에이전트

요구사항:
- 맥락 분석
   - 공황 원인
   - 사용자 감정 상태
   - 상황 심각성
- 의사소통
   - 실시간 음성 인식
   - 음성 출력
      
## Database
사용자 데이터 CRUD를 위한 데이터베이스
요구사항:
- 사용자 데이터 CRUD

## Networking
서비스 간의 네트워크 연결
요구사항:
- 프론트 서버와 백엔드 서버 연결
- 백엔드 서버와 DB 서버 연결
